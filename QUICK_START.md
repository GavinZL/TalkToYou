# 快速开始指南

欢迎使用 TalkToYou！本指南将帮助您快速上手这款智能对话应用。

## 📋 前置准备

### 必需项
- ✅ Mac电脑（macOS 12.0+）
- ✅ Xcode 14.0+
- ✅ iOS设备或模拟器（iOS 15.0+）
- ✅ 千问API密钥（从阿里云DashScope获取）

### 可选项
- 📦 SenseVoice Core ML模型文件

## 🚀 5分钟快速启动

### 步骤1：获取API密钥（2分钟）

1. 访问 https://dashscope.aliyun.com/
2. 注册/登录阿里云账号
3. 创建API-KEY并复制保存

### 步骤2：配置项目（1分钟）

```bash
# 克隆项目
git clone <repository-url>
cd TalkToYou

# 在Xcode中打开
open TalkToYou.xcodeproj
```

### 步骤3：添加模型（可选）

如果您有SenseVoice模型文件：
1. 将 `SenseVoice.mlmodelc` 拖入 `Resources` 目录
2. 确保勾选"Copy items if needed"

没有模型也可以运行，但ASR功能将返回模拟数据。

### 步骤4：运行应用（1分钟）

1. 在Xcode中选择目标设备
2. 按 `⌘R` 运行
3. 等待应用启动

### 步骤5：配置API（1分钟）

1. 在应用中进入"设置"标签
2. 粘贴您的API密钥
3. 点击"保存"

🎉 **完成！现在可以开始对话了！**

## 💬 开始第一次对话

### 方式1：语音对话
1. 回到"对话"标签
2. 点击底部的麦克风按钮
3. 开始说话
4. 再次点击停止录音
5. 等待AI回复并播放

### 方式2：文本对话
1. 在输入框中输入消息
2. 点击发送按钮
3. 等待AI回复并播放

## ⚙️ 基础设置

### 推荐配置

对于大多数用户，推荐以下配置：

```
API设置:
- 模型版本: qwen-turbo （速度快，成本低）

角色设定:
- 角色名称: 智能助手
- 角色描述: 你是一个友好的AI助手

语音设置:
- 语音类型: 中文 (zh-CN)
- 语速: 1.0 (正常语速，范围0-2.0)
- 音调: 1.0
- 音量: 1.0

对话设置:
- 上下文轮数: 10
- 生成温度: 0.8
- 最大tokens: 2000
```

### 个性化配置

**想要更快的响应？**
- 选择 qwen-turbo 模型
- 减少上下文轮数到 5
- 减少最大tokens到 1000

**想要更好的对话质量？**
- 选择 qwen-plus 或 qwen-max 模型
- 增加上下文轮数到 15-20
- 增加最大tokens到 3000-4000

**想要更有创意的回复？**
- 提高生成温度到 0.9-1.0

**想要更保守的回复？**
- 降低生成温度到 0.7

## 🎯 使用技巧

### 对话技巧
1. **明确表达** - 清晰地表述您的问题
2. **分步骤** - 复杂问题可以分步骤询问
3. **提供上下文** - 应用会记住最近的对话

### 录音技巧
1. **安静环境** - 在安静的环境中录音效果更好
2. **清晰发音** - 尽量清晰地发音
3. **适当距离** - 距离设备10-30cm最佳
4. **避免长时间** - 建议每次录音不超过30秒

### 节省费用
1. 使用 qwen-turbo 模型
2. 减少不必要的对话轮数
3. 定期清理历史记录
4. 监控API使用量

## 🔧 常见问题解决

### 问题1：录音无响应
**解决方案**：
1. 检查麦克风权限
2. 进入 iOS设置 → 隐私 → 麦克风
3. 确保TalkToYou已授权

### 问题2：API调用失败
**解决方案**：
1. 检查网络连接
2. 验证API密钥是否正确
3. 确认API账户有余额
4. 查看错误提示信息

### 问题3：语音识别不准确
**说明**：
- 当前版本ASR为示例实现
- 需要实际的SenseVoice模型文件
- 或者直接使用文本输入功能

### 问题4：TTS无声音
**解决方案**：
1. 检查设备音量
2. 检查静音开关
3. 在设置中调整音量参数
4. 重启应用
